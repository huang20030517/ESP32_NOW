# **ESP32-NOW 项目简要文档**

---

## **1. 项目简介**

ESP-NOW 是乐鑫推出的轻量级无线通信协议。它允许设备之间在不依赖 Wi-Fi 网络的情况下进行数据传输，非常适合低功耗、短距离的应用场景。

---

## **2. 功能概述**

- **双向通信**：设备之间点对点传输数据。
- **实时响应**：低延迟，适用于遥控和传感器数据采集。
- **简单部署**：无需路由器和复杂的网络设置。

---

## **3. 使用场景**

- **智能家居**：远程控制灯光、窗帘等设备。
- **传感器网络**：采集环境数据，实时上传。
- **工业监控**：设备之间传递状态信息和控制指令。

---

## **4. 注意事项**

- **MAC 地址配置**：添加正确的目标设备 MAC 地址。
- **数据长度限制**：每次发送数据包不得超过 250 字节。
- **信道设置**：确保通信设备使用相同的 Wi-Fi 信道。
